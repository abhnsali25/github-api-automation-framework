# 🚀 GitHub API Automation Framework

> ⚡ Production-ready API automation framework with CI/CD integration

## 📌 Overview

This project automates testing of GitHub APIs including:

- Create repository
- Get repository details
- Update repository
- Delete repository

---

## 🛠 Tech Stack

- Python
- Pytest
- Requests
- GitHub Actions (CI/CD)

---

## ▶️ Run Tests

pytest -v -s

---

## 🔄 CI/CD

Tests run automatically on every push using GitHub Actions.
